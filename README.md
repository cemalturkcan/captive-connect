# Captive Connect

Auto-connect to public WiFi networks on Android & iOS.

Detects captive portals, handles login automatically with saved credentials. Currently supports IBB WiFi (Istanbul public transit).

## Stack

- Kotlin Multiplatform + Compose Multiplatform
- Ktor Client for network requests
- Decompose for navigation
- Material3 dark theme
- EN/TR localization

## Build

```
./gradlew compileDebugSources
```
